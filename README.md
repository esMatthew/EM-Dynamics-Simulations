# EM-Dynamics-Simulations

These are a set of Physics simulations done to further understand the working mechanisms of diverse physical phenomena.

- **[Electromagnetism](https://github.com/esMatthew/EM-Dynamics-Simulations/tree/main/Dynamic%20Sims)** - Hand derived and simulated electro-statics, relativistic field compression, Lienard-Wiechert radiation

- **[Dynamics and Classical Mechanics](https://github.com/esMatthew/EM-Dynamics-Simulations/tree/main/Electromagnetic%20Sims)** - Single Pendulum, Double Pendulum, Mass-Spring-Damper system
